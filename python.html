Web Scraping Projects.

Below you will see three videos of different Projects I made.

The first project uses Jupyter Notebooks, Python, and Selenium to access information from a live website and return that information. It then displays to the screen the information we are asking for. After it displays the information it then turns the information into a bar chart and pie chart to help with visualization.

The second project also uses Python and Selenium but for the IDE it uses Spyder. This project collects data from indeed and turns it into a data set so we can collect and then save that information.

The third project will automatically run the indeed program at a set time everyday. After it runs it will send an email with the all of the data it collected.
